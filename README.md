## Featured Projects

### [Ornithopter for Medical Equipment Delivery](link-to-repo)
*SolidWorks, ROS2, Gazebo, 3D Printing*

Designed and simulated a flapping-wing medical drone capable of carrying 500g payloads. Secured **₹2,00,000 grant** from KTU Startup Cell for prototype development.

### [Amphibious Bot](link-to-repo)
*Fusion 360, ROS, SLAM, LiDAR*

Engineered a hybrid land-water robot with real-time SLAM mapping. Secured **₹1,30,000 grant** and published **IEEE research paper** at ACCESS '25.

### [Prosthetic Bionic Arm](link-to-repo)
*ESP32, Embedded C, SolidWorks, 3D Printing*

Developed an EMG-controlled prosthetic arm achieving **90% motion accuracy** in gesture-based control testing.

### [Vision-Based HMI System](link-to-repo)
*Python, OpenCV, MediaPipe*

Built a contactless home automation system using facial identification and gesture recognition. Achieved **95% recognition accuracy**.

### [AI Rehabilitation Therapy Assistant](link-to-repo) *(New!)*
*Python, MediaPipe, OpenCV, TensorFlow*

An AI-powered virtual physiotherapist that tracks arm movements via webcam and provides real-time feedback on exercise form, range of motion, and rep counting for rehabilitation patients.
